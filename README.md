# LDR情绪表达仪表板 💕

一个专为异地恋情侣设计的情绪表达和分享平台，帮助情侣们更好地沟通和理解彼此的情感状态。

![LDR情绪表达仪表板](https://img.shields.io/badge/LDR-情绪表达仪表板-ff6b9d?style=for-the-badge&logo=heart)

## ✨ 功能特色

### 🎭 情绪表达
- **8种情绪选择**: 开心、爱意满满、兴奋、平静、难过、想念、疲惫、焦虑
- **个性化消息**: 为每个情绪添加个人消息
- **实时同步**: 情绪状态实时更新显示
- **情绪动画**: 精美的动画效果增强用户体验

### 📊 数据统计
- **情绪分布图**: 可视化展示情绪趋势
- **互动统计**: 记录消息、拥抱、亲亲次数
- **时间筛选**: 支持今天、本周、本月数据查看
- **历史记录**: 完整的情绪交流历史

### 🎨 个性化设置
- **多主题支持**: 浅色、深色、浪漫三种主题
- **伴侣昵称**: 自定义伴侣显示名称
- **通知设置**: 可控制的消息通知
- **自动保存**: 数据自动本地保存

### 📱 现代化体验
- **PWA支持**: 可安装为手机应用
- **响应式设计**: 完美适配各种设备
- **离线使用**: 支持离线访问
- **快捷键支持**: 提高使用效率

## 🚀 快速开始

### 在线使用
直接在浏览器中打开 `index.html` 文件即可开始使用。

### 本地部署
1. 克隆或下载项目文件
2. 使用任意HTTP服务器运行项目
3. 推荐使用 `live-server` 进行开发：

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start
```

### 安装为PWA应用
1. 在支持PWA的浏览器中打开应用
2. 点击地址栏的"安装"按钮
3. 应用将添加到主屏幕，可离线使用

## 🎯 使用指南

### 基本操作
1. **选择情绪**: 点击情绪按钮选择当前心情
2. **添加消息**: 在文本框中输入想对Ta说的话
3. **发送情绪**: 点击发送按钮分享你的情绪
4. **回应伴侣**: 使用拥抱和亲亲按钮回应Ta的情绪

### 快捷键
- `Ctrl/Cmd + Enter`: 发送情绪
- `Ctrl/Cmd + 1-8`: 快速选择情绪
- `F1`: 显示帮助信息

### 数据管理
- **导出数据**: 在设置中点击"导出数据"保存所有记录
- **导入数据**: 使用"导入数据"功能恢复之前的记录
- **清空历史**: 可以清空所有历史记录重新开始

## 📊 数据存储

应用使用浏览器的 `localStorage` 进行数据存储，包括：

- **情绪历史记录**: 所有发送和接收的情绪
- **用户设置**: 主题、昵称等个人偏好
- **统计数据**: 互动次数和情绪分布

## 🔒 隐私保护

- **本地存储**: 所有数据仅存储在用户设备上
- **无服务器**: 不需要注册账号或上传数据
- **完全私密**: 情绪记录完全属于用户个人

## 🔧 自定义开发

### 添加新情绪
在 `script.js` 中的情绪数组中添加新的情绪选项：

```javascript
const newMoods = [
    { mood: 'custom', emoji: '😎', label: '自定义情绪' }
];
```

### 修改主题色彩
在 `styles.css` 的 CSS 变量中修改颜色：

```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
}
```

### 扩展统计功能
可以在统计模块中添加更多数据分析功能。
